## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/alissoncavalcanticma/node_uploadfile_api

# Access
$ cd node_uploadfile_api

# Install dependencies
$ npm i

# Run the project
$ npm run start-dev

# The server will initialize in the <http://localhost:4000>
```
